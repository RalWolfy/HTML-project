HTML is a very frendly language and is on good terms with almost all of the web browsers. HTML is a form of markup aimed at presenting text documents on a single page using specialized rendering software called an HTML user agent, the best example of such software being the web browser. It is the least complicated when it comes to search engines.

Why isn't HTML a programming language?

It controls the presentation, structure and layout of the data on a webpage. It just does not contain any functional or programming logic. Manipulating data is not in its nature, event handling, taking input and displaying output. It just does not contain any functional or programming logic.

What is a programming language?
A programming language is a valid set of expressions and rules (or techniques) for formulating instructions for a "computer". A programming language has a defined set of semantic and syntactic rules. It allows the programmer to specify exactly and announced actions that the computer must perform, in order with data. The specification basically consists in drawing up/writing the necesary programs (programming).
What is metadata?
Metadata are data who describe other data, such as paragraphs. A colection of data.
