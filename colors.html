Colors

   We can set the background, text color, border color. In HTML, colors can be specified using a color name or RGB , HEX , HSL , RGBA , HSLA values.

Color

Text color

Border color

RGBA and HSLA

    RGBA and HSLA color values are an extension of RGB and HSL color values with an Alpha channel-which specififies the opacity for a color. The alpha parameter is a number between 0.0 fully transparent and 1.0 (not transparent at all).

   <font color="hsl(114, 30%, 60%, 0.4)">

Shades of grey

   Shades of grey are often defined using equal values for all three parametres.

Hue

   Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 360 is blue.

Saturation

   Saturation is a percentage value. 0 means a shade of gray, and " 100% is the full color.

Lightness

   Lightness is also a percentage value. 0 is black, and 100 white.
