Lists

          Lists can be of 2 types: ordered and unordered. This can be styled in CSS or personalized. Also, we have list of definitions.

The ordered ones can be:

Type="1" 

<ol type="1">
<li> The first </li>
<li> The second </li>
<li> The third </li>
</ol>
  1. The first
  2. The second
  3. The third

     By default, an ordered list will start counting from 1. If we want to start counting from a specified number, we can use the start attribute:

Control List Counting 

<ol type="1" start="5">
<li> The first </li>
<li> The second </li>
<li> The third </li>
</ol>
  1. The first
  2. The second
  3. The third

Navigation Menu

We can use CSS to style the list horizontally, to create a navigation menu:

  • Home
  • News
  • Contact
  • About
 

<ul>
  <li><a href="#home">Home</a></li>
  <li><a href="#news">News</a></li>
  <li><a href="#contact">Contact</a></li>
  <li><a href="#about">About</a></li>
</ul>
In CSS 

<style>
ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: #333333;
}

li {
  float: left;
}

li a {
  display: block;
  color: white;
  text-align: center;
  padding: 16px;
  text-decoration: none;
}

li a:hover {
  background-color: #111111;
}
</style>>
